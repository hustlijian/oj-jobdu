题目：题目1010：A + B
-----------
时间限制：1 秒内存限制：32 兆特殊判题：否提交：3816解决：2012

题目描述：
-----------
读入两个小于100的正整数A和B,计算A+B.
需要注意的是:A和B的每一位数字由对应的英文单词给出.

输入：
-----------
测试输入包含若干测试用例,每个测试用例占一行,格式为"A + B =",相邻两字符串有一个空格间隔.当A和B同时为0时输入结束,相应的结果不要输出.

输出：
-----------
对每个测试用例输出1行,即A+B的值.

样例输入：
-----------
one + two =  
three four + five six =  
zero seven + eight nine =  
zero + zero =  

样例输出：
-----------
3  
90  
96

-----------
2005年浙江大学计算机及软件工程研究生机试真题
