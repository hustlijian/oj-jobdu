题目：题目1083：特殊乘法
-----------
时间限制：1 秒内存限制：32 兆特殊判题：否提交：1468解决：1003

题目描述：
-----------
写个算法，对2个小于1000000000的输入，求结果。
特殊乘法举例：123 * 45 = 1*4 +1*5 +2*4 +2*5 +3*4+3*5

输入：
-----------
 两个小于1000000000的数

输出：
-----------
输入可能有多组数据，对于每一组数据，输出Input中的两个数按照题目要求的方法进行运算后得到的结果。

样例输入：
-----------
123 45

样例输出：
-----------
54

-----------
2010年清华大学计算机研究生机试真题
