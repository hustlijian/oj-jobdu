题目：英雄会第四届在线编程大赛·线上初赛：带通配符的数
-----------

题目描述：
-----------
给定一个带通配符问号的数W，问号可以代表任意一个一位数字。
再给定一个整数X，和W具有同样的长度。
问有多少个整数符合W的形式并且比X大？

输入格式
多组数据，每组数据两行，第一行是W，第二行是X，它们长度相同。在[1..10]之间.
输出格式
每行一个整数表示结果。

输入：
-----------
多组数据，每组数据两行，第一行是W，第二行是X，它们长度相同。在[1..10]之间.

输出：
-----------
每行一个整数表示结果。

样例输入：
-----------
36?1?8
236428
8?3
910
?
5

样例输出：
-----------
100
0
4
